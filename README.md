# KML轨迹可视化工具

这是一个简单的Web应用程序，用于可视化KML/KMZ文件中的轨迹数据。

## 功能特点

- 支持上传KML和KMZ文件
- 在交互式地图上显示轨迹
- 显示起点和终点标记
- 显示轨迹点数量统计
- 支持中文界面

## 安装要求

1. Python 3.7+
2. 安装依赖包：
```bash
pip install -r requirements.txt
```

## 运行方法

在命令行中执行：
```bash
streamlit run app.py
```

## 使用说明

1. 启动应用后，在浏览器中打开显示的地址
2. 点击"选择KML/KMZ文件"按钮上传文件
3. 等待文件解析完成，地图将自动显示轨迹
